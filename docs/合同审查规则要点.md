# 文件审查要点

本文仅列「审什么」：劳动合同与民事起诉状各自要对照检查的要点。便于业务上对齐标准，与系统实现无关。

---

## 一、劳动合同审查要点

### 主要依据（观念上应对照）

- 《中华人民共和国劳动合同法》
- 《劳动合同法实施条例》

### 主体

- 年龄是否达到法定就业年龄（年满十六周岁）。
- 与岗位相关的学历、资格等材料是否需审查、是否约定。
- 与原单位劳动关系是否解除：离职证明等。
- 身体状况：体检等是否需审查、是否约定。
- 是否具备劳动行为能力（能否独立行使劳动权利、履行劳动义务）。
- 是否属于特殊用工情形：外籍人员、在校实习生、兼职、退休返聘等。

### 合同期限与试用期

- 是否约定合同期限及起止时间。
- 是否约定试用期；试用期长度是否符合法律对合同期限档次的限制；同一用人单位与同一劳动者只能约定一次试用期；短期合同、以完成一定工作任务为期限的合同是否依法不得约定或限制试用期；外籍人员合同期限是否超过法定上限；任务型合同是否写明任务起止时间。

  **补充注意：** 三个月以上不满一年的，试用期不得超过一个月；一年以上不满三年的，不得超过两个月；三年以上固定期限和无固定期限的，不得超过六个月；三个月以下及任务型合同不得约定试用期。

- 是否约定试用期内「不符合录用条件」的具体情形。

### 工作性质、时间、地点、内容

- 工作地点是否约定；是否约定单位可调整地点及调整条件。
- 工作岗位、岗位职责是否约定；是否约定单位可单方调岗及条件。
- 工时制度是否明确（标准工时、综合计算工时、不定时工时等）。
- 加班是否需书面申请与审批。
- 休假是否约定；是否有优于年休假的福利假；福利假与年休假未休完如何处理（作废或补偿等）；是否建议通过规章制度进一步细化。

### 劳动报酬与保险福利

- 试用期工资是否约定；是否不低于转正工资的百分之八十且不低于当地最低工资标准。
- 转正后工资是否约定。
- 支付方式、支付时间是否约定。
- 奖金是否约定及计发规则。
- 保险福利待遇是否约定。

### 劳动纪律、劳动保护、劳动条件与职业危害

- 劳动纪律是否明确。
- 违反劳动纪律的后果是否明确。
- 存在职业病风险的行业是否约定职业危害防护。

### 权利义务（保密与竞业等）

- 保密义务、保密信息范围、保密期限、保密措施、违反保密的赔偿责任是否约定。
- 职务成果范围、知识产权申报协助、非职务成果单位优先受让等是否约定。
- 是否约定另行签订竞业限制协议；竞业限制范围、期限、补偿金是否约定。

### 变更、续订、解除与终止

- 合同变更情形及单位是否有权单方变更是否明确。
- 续订情形是否明确。
- 解除、终止情形是否明确。
- 解除或终止后劳动者义务（如服务期与违约金）是否明确。

### 经济补偿与赔偿

- 劳动者向单位赔偿损失的情形是否约定。
- 单位支付经济补偿的情形是否约定。
- 经济补偿计算标准是否约定。

### 争议处理与其他

- 劳动争议解决方式是否约定；是否可约定先行调解。

  **补充注意：** 劳动争议依法应先申请劳动仲裁，对仲裁裁决不服再向人民法院起诉；宜约定向有管辖权的劳动人事争议仲裁委员会申请仲裁。

- 合同生效时间是否约定。
- 补充协议效力是否约定。
- 劳动者送达地址、电子邮箱、联系电话是否约定；变更送达方式是否约定告知义务。
- 签章栏是否具备。

## 二、民事起诉状审查要点

### 主要依据（观念上应对照）

- 《中华人民共和国民事诉讼法》关于起诉条件的规定。
- 《中华人民共和国民事诉讼法》关于起诉状应记明事项的规定。

### 当事人与诉讼主体

- 原告：姓名或名称、性别、住所、身份证明或统一社会信用代码等信息是否齐全、与主体类型是否匹配。
- 被告：姓名或名称、住所等能否特定被告身份。
- 共同诉讼：必要共同诉讼的当事人是否列全；普通共同诉讼合并起诉的理由是否说明。
- 无民事行为能力人、限制民事行为能力人是否列明法定代理人及与原告的关系。
- 委托诉讼代理人：姓名、单位、职务、代理权限（一般授权与特别授权）是否写明。
- 应参加诉讼的第三人是否列明。

### 诉讼请求

- 请求是否具体、明确，便于审理与执行。
- 给付请求：标的、种类、数额或计算方法是否写明（如本金、利息、违约金、起止时间等）。
- 确认之诉、形成之诉：法律关系是否表述清楚。
- 多项请求之间是否矛盾、是否需分案或择一主张。
- 是否请求诉讼费用；保全费、鉴定费等是否需单列。
- 保全、先予执行等申请与正文是否一致。

### 事实与理由

- 事实是否按时间或逻辑顺序叙述清楚。
- 合同订立、履行、违约或侵权等关键事实及因果关系是否写明。
- 是否区分事实陈述与法律评价，避免空洞结论代替事实。
- 主要法律依据是否概括引用。
- 协商、催告、仲裁或行政处理等前置经过是否说明（案情需要时）。
- 理由是否逐项回应诉讼请求、覆盖请求权基础。

### 证据与举证

- 是否附证据目录或逐项列明证据名称、来源、证明目的。
- 证据与待证事实是否对应。
- 电子数据、视听资料等是否说明来源与形式（需要时）。
- 申请证人出庭的，是否列明证人及拟证明事实。
- 举证责任、对方持有证据的申请等是否说明（适用时）。

### 管辖与程序

- 致送法院全称是否与级别管辖、专门管辖相符。
- 地域管辖依据是否说明（被告住所地、合同履行地、侵权行为地、协议管辖等）。
- 依法须先仲裁的劳动争议等，是否说明仲裁经过或一并说明。
- 诉讼时效是否提示或说明起算、中断（适用时）。
- 是否涉及重复起诉、已生效裁判等需排除的情形（适用时）。

### 文书形式与结尾

- 「此致」、受诉法院全称、具状人签章位置是否具备。
- 起诉日期是否写明。
- 副本份数、提交材料是否符合法院要求。
- 送达地址、电话、电子邮箱是否注明（法院要求时）。
- 法人起诉的法定代表人或负责人签名、盖章是否完整。
